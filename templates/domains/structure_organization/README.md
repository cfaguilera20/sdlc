# Structure & Organization Domain

This domain covers all aspects of organizational structure, hierarchy, positions, and organizational units.

## Domain Overview

The Structure & Organization domain manages the complete organizational structure, including:
- Organizational hierarchy and tree structure
- Areas, departments, and organizational units
- Position definitions and management
- Organizational levels and profiles
- Company and group structures
- Locality and geographic organization

## Domain Expert Agents

1. **Domain Expert — Structure & Organization (Hierarchy & Units)**
   - Focus: Organizational tree, areas, departments, organizational levels, company structure
   - Template: `domain_expert_structure_hierarchy_units.md`

2. **Domain Expert — Structure & Organization (Positions & Profiles)**
   - Focus: Position definitions, job profiles, position assignments, profile management
   - Template: `domain_expert_structure_positions_profiles.md`

## Usage

When working on organizational structure-related tickets:

1. Identify which subdomain(s) the ticket touches
2. Run the appropriate Domain Expert agent(s) using the template files
3. Collect the `DomainKnowledgePack` JSON output(s)
4. Use the pack(s) as input to Agent 02 (Product Analyst) for backlog generation

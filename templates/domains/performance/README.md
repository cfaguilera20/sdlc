# Performance Domain

This domain covers all aspects of performance management, evaluations, objectives, and KPIs.

## Domain Overview

The Performance domain manages the complete performance management lifecycle, including:
- Performance evaluations and reviews
- Objective setting and tracking
- KPI management and measurement
- Performance tasks and workflows
- Participant management
- Performance reporting and analytics

## Domain Expert Agents

1. **Domain Expert — Performance (Evaluations & Reviews)**
   - Focus: Evaluation cycles, review processes, evaluation phases, participant management
   - Template: `domain_expert_performance_evaluations.md`

2. **Domain Expert — Performance (Objectives & KPIs)**
   - Focus: Objective creation, KPI definitions, tracking, measurement, alignment
   - Template: `domain_expert_performance_objectives_kpis.md`

3. **Domain Expert — Performance (Workflows & Tasks)**
   - Focus: Performance workflows, task management, phase transitions, approvals
   - Template: `domain_expert_performance_workflows_tasks.md`

## Usage

When working on performance-related tickets:

1. Identify which subdomain(s) the ticket touches
2. Run the appropriate Domain Expert agent(s) using the template files
3. Collect the `DomainKnowledgePack` JSON output(s)
4. Use the pack(s) as input to Agent 02 (Product Analyst) for backlog generation

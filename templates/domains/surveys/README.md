# Surveys Domain

This domain covers all aspects of employee surveys, psychometric assessments, and NOM-035 compliance.

## Domain Overview

The Surveys domain manages the complete survey lifecycle, including:
- Survey creation and configuration
- Psychometric assessments
- NOM-035 compliance surveys
- Survey evaluations and results
- Participant management
- Survey reporting and analytics

## Domain Expert Agents

1. **Domain Expert — Surveys (Surveys & Assessments)**
   - Focus: Survey creation, psychometric assessments, NOM-035 surveys, survey configuration
   - Template: `domain_expert_surveys_assessments.md`

2. **Domain Expert — Surveys (Evaluations & Results)**
   - Focus: Survey evaluations, result collection, participant management, reporting
   - Template: `domain_expert_surveys_evaluations_results.md`

## Usage

When working on survey-related tickets:

1. Identify which subdomain(s) the ticket touches
2. Run the appropriate Domain Expert agent(s) using the template files
3. Collect the `DomainKnowledgePack` JSON output(s)
4. Use the pack(s) as input to Agent 02 (Product Analyst) for backlog generation
